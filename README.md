# SPI using Verilog

## Overview

This project implements the Serial Peripheral Interface (SPI) protocol using Verilog HDL. The design supports high-speed synchronous serial communication between a master and slave device.

---

## Features

- SPI Master
- SPI Slave
- MOSI
- MISO
- Clock Generation
- Chip Select Logic

---

## Technologies Used

- Verilog HDL
- ModelSim / QuestaSim
